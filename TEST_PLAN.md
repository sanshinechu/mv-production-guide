## 測試執行步驟

1. **本地開發環境**
   ```bash
   cd mv-guide
   npm install --legacy-peer-deps
   npm run dev
   ```

2. **訪問應用**
   - 打開 http://localhost:5173

3. **執行測試清單**
   - 按順序測試每個頁面
   - 記錄任何錯誤或異常

4. **構建和預覽**
   ```bash
   npm run build
   npm run preview
   ```

5. **部署到 Vercel**
   - 確認構建成功
   - 驗證部署後的應用
